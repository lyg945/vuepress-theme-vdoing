## 快速上手

```bash
# clone the project
git clone https://github.com/lyg945/vuepress-theme-vdoing

# enter the project directory
cd vuepress-theme-vdoing

# install dependency
npm install # or yarn install

# develop
npm run dev # or yarn dev
```
